# CRUD API

O projeto é um C.R.U.D. para criação de livros com armazenamento em memória.

## Requisitos

Go (versão >= 1.22)

Não há dependências externas.

## Rodando a aplicação

A API foi iniciada como localhost porta 8080.

com o comando abaixo a aplicação será iniciada.

``` bash
go run .
```
